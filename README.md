# Bubble Chat Copy

Copy the user's chat bubble replying to your message.

> [!WARNING]
> This script uses an unofficial API wrapper for Amino

## How to use

- Download this script. (See [Installation](#Installation))
- Setup the environment variables. (See [Setup](#Setup))
- Run the script. (See [Running](#Running))

## Installation

### Git
```
git clone https://github.com/ViktorSky/amino-copy-bubble.git
```

## Setup

Set all parameter values in `.env` file

## Running

Run with the python command and open the Amino Application.
```
python main.py
```
Reply the message with the bubble to copy.
Confirm the copy request.
